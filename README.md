# InternetProgramming

## 📌 Project Title
Simple E-Commerce Shopping Cart Web Application

## 💻 Languages & Technologies Used
HTML5 – Structure of the website
CSS3 – Styling and layout
JavaScript (Vanilla JS) – Cart logic and dynamic updates
Node.js – Backend server
MySQL – Database management

## ⚠️ Challenges & Solutions
One of the main challenges was managing the cart state dynamically using JavaScript. Initially, updating quantities did not reflect correctly in the UI. This was solved by re-rendering the cart every time the data changes. Another issue was handling edge cases like decreasing quantity below one, which required proper validation logic. Additionally, keeping the code simple while maintaining functionality was challenging but achieved by using basic arrays and functions. Overall, the project improved understanding of DOM manipulation and event handling.

## 📖 Project Summary
This project is a simple e-commerce shopping cart web application developed for an Internet Programming course. It allows users to browse products, add items to a cart, update quantities, and manage their selections dynamically.
The frontend is built using HTML, CSS, and JavaScript, focusing on DOM manipulation and event handling to provide real-time updates without page reloads. The backend uses Node.js and MySQL to handle product data and support basic server-side operations.
Key features include:
# Add/remove products from cart
# Update item quantities dynamically
# Real-time cart total calculation
# Basic database integration for product storage
The project demonstrates core web development concepts such as client-server interaction, state management, and database connectivity, while keeping the implementation simple and beginner-friendly.

## 🚀 How to Run
1. Download or clone the repository  
2. Open the project folder  
3. Double-click **index.html**  
4. The website will run in your browser  

Troubleshooting:
1. Go to dir
2. Run cmd: 
    mysql -u root -p < db.sql
    npm run install
    node server.js
3. ready to go
